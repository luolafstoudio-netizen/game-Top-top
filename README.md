## 🕹 How to Play

</div>

### Step 1 — Choose Your Way to Play

**Option A — Online (GitHub Pages)**
```
1. Visit the game link on GitHub Pages
2. No download required
3. The game loads instantly in your browser
```

**Option B — Offline (Downloaded)**
```
1. Download the project files from this repository
2. Open survival_game.html in any modern browser
3. Play without any internet connection
```

---

### Step 2 — Learn the Controls

| Input | Action |
|---|---|
| `W A S D` or `Arrow Keys` | Move your character |
| `Mouse Click` | Attack enemies / Interact with objects |
| `H` | Return to Safe House instantly |
| `C` | Open Crafting Bench |
| `I` | Open Inventory / Chest |
| `D-Pad` *(mobile)* | Move character on touch screens |

---

### Step 3 — Survive and Grow

```
🍎 Eat to stay alive     →   Hunger bar keeps you going
😴 Sleep to recover      →   Stamina and HP restore at night
⚒️ Craft to get stronger →   Axe, Sword, Bow, Walls, and more
🌲 Explore the forest    →   Resources and danger both live there
🧟 Fight enemies         →   Every kill earns EXP and drops loot
⭐ Level up              →   Each level makes you permanently stronger
```
